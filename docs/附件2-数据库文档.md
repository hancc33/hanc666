**关联关系说明**：
- Video 与 VideoScript：一对多关系（一个视频可对应多个脚本）
- 其他实体之间为独立关系

## 二、数据库表结构设计

### 1. 管理员表（admin）

| 字段名 | 数据类型 | 长度 | 主键/外键 | 允许为空 | 注释 |
|-------|---------|------|----------|---------|------|
| id | BIGINT | - | 主键 | 否 | 主键ID，自增 |
| username | VARCHAR | 50 | - | 否 | 管理员账号，唯一 |
| password | VARCHAR | 255 | - | 否 | 密码（BCrypt加密） |
| nickname | VARCHAR | 50 | - | 是 | 昵称 |

### 2. 用户表（app_user）

| 字段名 | 数据类型 | 长度 | 主键/外键 | 允许为空 | 注释 |
|-------|---------|------|----------|---------|------|
| id | BIGINT | - | 主键 | 否 | 主键ID，自增 |
| username | VARCHAR | 50 | - | 否 | 用户名，唯一 |
| password | VARCHAR | 255 | - | 否 | 密码（BCrypt加密） |
| nickname | VARCHAR | 100 | - | 是 | 昵称 |
| email | VARCHAR | 200 | - | 是 | 邮箱 |
| create_time | DATETIME | - | - | 否 | 注册时间 |

### 3. Banner表（banner）

| 字段名 | 数据类型 | 长度 | 主键/外键 | 允许为空 | 注释 |
|-------|---------|------|----------|---------|------|
| id | BIGINT | - | 主键 | 否 | 主键ID，自增 |
| title | VARCHAR | 200 | - | 是 | 标题 |
| subtitle | VARCHAR | 1000 | - | 是 | 副标题 |
| image_url | VARCHAR | 500 | - | 否 | 图片地址 |
| description | VARCHAR | 2000 | - | 是 | 描述 |
| sort_order | INT | - | - | 是 | 排序 |
| active | BOOLEAN | - | - | 是 | 是否启用 |
| create_time | DATETIME | - | - | 否 | 创建时间 |

### 4. 视频表（video）

| 字段名 | 数据类型 | 长度 | 主键/外键 | 允许为空 | 注释 |
|-------|---------|------|----------|---------|------|
| id | BIGINT | - | 主键 | 否 | 主键ID，自增 |
| title | VARCHAR | 200 | - | 否 | 视频名称 |
| description | VARCHAR | 2000 | - | 是 | 视频简介 |
| video_url | VARCHAR | 500 | - | 否 | 视频文件地址 |
| cover_url | VARCHAR | 500 | - | 是 | 封面图片地址 |
| category | VARCHAR | 50 | - | 是 | 分类（实践记录/活动宣传/成果展示/采访记录） |
| publish_time | DATETIME | - | - | 是 | 发布时间 |
| create_time | DATETIME | - | - | 否 | 创建时间 |

### 5. 视频脚本表（video_script）

| 字段名 | 数据类型 | 长度 | 主键/外键 | 允许为空 | 注释 |
|-------|---------|------|----------|---------|------|
| id | BIGINT | - | 主键 | 否 | 主键ID，自增 |
| title | VARCHAR | 200 | - | 否 | 脚本标题 |
| description | VARCHAR | 2000 | - | 是 | 脚本描述 |
| file_url | VARCHAR | 500 | - | 否 | 文件地址 |
| file_type | VARCHAR | 50 | - | 是 | 文件类型（doc/docx/pdf/txt） |
| video_id | BIGINT | - | 外键 | 是 | 关联视频ID |
| create_time | DATETIME | - | - | 否 | 创建时间 |

### 6. 活动图片表（activity_image）

| 字段名 | 数据类型 | 长度 | 主键/外键 | 允许为空 | 注释 |
|-------|---------|------|----------|---------|------|
| id | BIGINT | - | 主键 | 否 | 主键ID，自增 |
| title | VARCHAR | 200 | - | 否 | 图片标题 |
| description | VARCHAR | 500 | - | 是 | 图片描述 |
| image_url | VARCHAR | 500 | - | 否 | 图片地址 |
| category | VARCHAR | 50 | - | 是 | 分类（志愿服务/课堂活动/团队合影/实践过程） |
| sort_order | INT | - | - | 是 | 排序 |
| create_time | DATETIME | - | - | 否 | 上传时间 |

### 7. 留言表（message）

| 字段名 | 数据类型 | 长度 | 主键/外键 | 允许为空 | 注释 |
|-------|---------|------|----------|---------|------|
| id | BIGINT | - | 主键 | 否 | 主键ID，自增 |
| username | VARCHAR | 50 | - | 否 | 留言用户 |
| title | VARCHAR | 200 | - | 否 | 留言标题 |
| content | VARCHAR | 2000 | - | 否 | 留言内容 |
| reply | VARCHAR | 1000 | - | 是 | 管理员回复 |
| replied | BOOLEAN | - | - | 是 | 是否已回复 |
| create_time | DATETIME | - | - | 否 | 留言时间 |

### 8. 实践资料表（material）

| 字段名 | 数据类型 | 长度 | 主键/外键 | 允许为空 | 注释 |
|-------|---------|------|----------|---------|------|
| id | BIGINT | - | 主键 | 否 | 主键ID，自增 |
| title | VARCHAR | 200 | - | 否 | 资料标题 |
| description | VARCHAR | 2000 | - | 是 | 资料描述 |
| file_url | VARCHAR | 500 | - | 否 | 文件地址 |
| file_type | VARCHAR | 50 | - | 是 | 文件类型 |
| category | VARCHAR | 50 | - | 是 | 分类（实践报告/视频脚本/成果材料/其他） |
| create_time | DATETIME | - | - | 否 | 上传时间 |

### 9. 网站设置表（site_setting）

| 字段名 | 数据类型 | 长度 | 主键/外键 | 允许为空 | 注释 |
|-------|---------|------|----------|---------|------|
| id | BIGINT | - | 主键 | 否 | 主键ID，自增 |

| setting_value | VARCHAR | 5000 | - | 是 | 设置值 |
| description | VARCHAR | 100 | - | 是 | 设置说明 |
