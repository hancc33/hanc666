# 附件1：系统介绍及功能清单

## 一、系统概述

本系统旨在为泉州职业技术大学八方书院"助苗·助梦·护苗"暑期社会实践活动提供一个成果展示与信息管理平台，基于SpringBoot、Spring Data JPA、MySQL/H2、Vue3、Element Plus等技术搭建，具备高稳定性、前后端分离、易扩展等特点，可满足大学生志愿者团队在社会实践活动中的内容展示、视频管理、图片管理、留言互动等业务需求。

**开发背景**：大学生暑期社会实践活动是高校育人的重要环节，"助苗·助梦·护苗"实践队围绕帮扶困境儿童、青少年成长开展了一系列活动，需要一个专业的展示平台来记录和呈现实践成果。

**核心问题**：传统实践活动成果展示依赖线下展板或简单的PPT演示，缺乏系统化、可交互的展示方式，也不便于管理员自主更新内容。

**目标用户**：
- 普通访客：浏览实践成果、观看视频、留言互动
- 注册用户：登录后可发表留言
- 管理员：管理网站所有内容（视频、图片、资料、留言等）

## 二、功能清单

| 功能模块 | 功能名称 | 功能描述 | 功能特点 |
|---------|---------|---------|---------|
| 用户系统 | 用户注册 | 游客可注册成为用户 | 支持用户名、昵称、邮箱 |
| 用户系统 | 用户登录 | 用户登录获取JWT令牌 | Token认证，安全可靠 |
| 用户系统 | 用户登出 | 用户退出登录 | 清除本地Token |
| 首页展示 | Banner轮播 | 首页顶部Banner展示 | 管理员可自定义 |
| 首页展示 | 统计数据 | 展示视频、图片、留言数量 | 实时统计 |
| 首页展示 | 内容预览 | 视频、图片、留言预览 | 快速了解实践内容 |
| 视频管理 | 视频上传 | 上传MP4格式视频 | 支持大文件上传 |
| 视频管理 | 视频播放 | 在线播放视频 | HTML5原生播放器 |
| 视频管理 | 视频分类 | 按类别筛选视频 | 实践记录、活动宣传等 |
| 视频管理 | 分页查询 | 视频列表分页展示 | 支持多条件组合查询 |
| 视频脚本 | 脚本上传 | 上传Word/PDF/TXT脚本 | 绑定对应视频 |
| 视频脚本 | 脚本查看 | 在线查看脚本内容 | PDF预览、TXT在线阅读 |
| 视频脚本 | 脚本下载 | 下载脚本文件 | 支持多种格式 |
| 图片管理 | 图片上传 | 上传活动照片 | 支持JPG/PNG等格式 |
| 图片管理 | 图片展示 | 图片墙+大图预览 | 分类筛选、悬浮效果 |
| 图片管理 | 图片分类 | 按类别筛选图片 | 志愿服务、课堂活动等 |
| 资料管理 | 资料上传 | 上传实践报告、成果材料 | 支持多种文档格式 |
| 资料管理 | 资料下载 | 下载资料文件 | 一键下载 |
| 留言互动 | 发表留言 | 注册用户发表留言 | 标题+内容 |
| 留言互动 | 留言展示 | 展示留言列表 | 按时间倒序 |
| 留言互动 | 管理员回复 | 管理员回复留言 | 可查看回复状态 |
| 实践成果 | 实践介绍 | 展示背景、目标、对象、意义 | 内容可后台配置 |
| 实践成果 | 活动内容 | 分类展示安全教育、成长陪伴 | 结构化展示 |
| 实践成果 | 创新成果 | 展示安全小卫士卡牌 | 桌游卡牌介绍 |
| 管理后台 | 控制台 | 数据统计、快速操作 | 一目了然 |
| 管理后台 | Banner管理 | 增删改查Banner | 排序、启用/禁用 |
| 管理后台 | 网站设置 | 配置网站基本信息 | 名称、介绍、实践信息 |
